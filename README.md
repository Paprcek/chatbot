# 🛡️ Chatbot v2: Duchovní Debugger & Biblický Průvodce

Vítejte v projektu, který propojuje analytický svět IT s hloubkou biblické moudrosti. Tento chatbot je navržen jako nástroj pro identifikaci „duchovního malware“, analýzu toxické komunikace a hledání vnitřního míru v digitálním chaosu.

## ✨ Klíčové funkce
- **Bible RAG (Retrieval-Augmented Generation):** Inteligentní vyhledávání v českém překladu Bible (B21).
- **Cloud Intelligence:** Využívá nejmodernější modely (Llama 3.3 / Google Gemini) přes API pro rychlé a moudré odpovědi.
- **Vizuální analýza:** Generování Mermaid diagramů pro vizualizaci komunikačních smyček a logických struktur.
- **IT-Spiritual Terminologie:** Debugování víry, vnitřní firewally a optimalizace duševního klidu.

## 🛠️ Tech Stack
- **Jazyk:** Python 3.11 (stabilní základ)
- **Správce balíčků:** [uv](https://github.com/astral-sh/uv) (extrémně rychlý a efektivní)
- **AI Engine:** Groq API / Google Generative AI
- **Embeddings:** `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2`

## 🚀 Rychlý start

1. **Instalace prostředí:**
   ```bash
   uv sync

2. **Nastavení klíčů:**
    GROQ_API_KEY=váš_klíč
    GOOGLE_API_KEY=váš_klíč

3. **Spuštění:**
    ```bash
    uv run brain_v2.py

## 📖 O projektu
Tento projekt je součástí tvorby knihy o duchovním zneužívání a efektivní komunikaci. Cílem je naučit uživatele včas identifikovat „nekonečné smyčky“ neplodných diskuzí a chránit svůj vnitřní mír.
